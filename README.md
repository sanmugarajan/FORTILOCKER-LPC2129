🔐 FORTILOCKER – Secure 2-Step Authentication Based Bank Locker Access Control System:-

FORTILOCKER is an embedded security system designed to provide secure access control for bank lockers using Two-Factor Authentication (2FA).

The system verifies the user through:

4-Digit PIN Authentication
OTP (One-Time Password) Verification via GSM

Only after successful verification is access granted to the locker through a DC motor-based locking mechanism.

Features:

4-Digit PIN Verification,
 GSM-Based OTP Authentication,
 Two-Factor Authentication (2FA),
 LCD User Interface,
 Matrix Keypad Input,
 Buzzer Alert for Unauthorized Access,
 DC Motor Controlled Locker Mechanism,
 Maximum Attempt Limitation for Enhanced Security,
 Hardware Components,
 LPC2129 ARM7 Microcontroller,
 SIM800C GSM Module,
 16x2 LCD Display,
 4x4 Matrix Keypad,
 DC Motor,
 Buzzer,
 Power Supply Unit,
 Software Tools,
 Keil µVision,
 Proteus Professional,
 Embedded C,

Project Workflow:

User Entry → PIN Verification → OTP Generation → OTP Sent via GSM → OTP Verification → Locker Access Granted → Motor Operated Locker

Simulation:

The system logic was validated using Proteus simulation with LPC2124 before hardware implementation.

Hardware Implementation:

The project was implemented and tested on LPC2129 ARM7 hardware.

Applications:

Bank Locker Security,
 Personal Safe Lockers,
 Restricted Access Storage Systems,
 Secure Document Cabinets
